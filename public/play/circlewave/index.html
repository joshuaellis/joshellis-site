Circle Wave

Inspired by Mike Bostock's circle wave observable using D3, I sought to replicate this as a means to further learn Canvas & JS 2D animation but at the same time remove the D3 dependencies. During the process I created two other modes, dots & lines which then creates a different feel and using the same parameters changes the visual entirely.
